Statistic Analyzer
=================================
Statistic Analyzer는 지정 라벨을 이용하여 시계열 데이터를 분석하는 모듈이다. 
분석 기준과 분석된 정보를 활용하여 시각화 자료를 생성할 수 있다.시각화 자료는 CLUST 플랫폼 Feature Exploration에서 제공한다.
